# 目录介绍

> src目录下的顺序就是学习顺序

- dist--------------------ts编译后的js代码
- docs-------------------说明文档
- skill--------------------学习中总结的小技巧
- src---------------------ts源码
    - types---------------声明文件
        - dataType---------数据类型
        - function---------函数
        - class------------类
        - class_extends----类的继承（及原理）
        - class_decorators-类装饰器
        - class_abstracts--抽象类
        - interface--------接口
        - generics---------泛型
        - compatible-------兼容性
        - protect----------类型保护
        - change-----------类型变换
        - module-----------模块
